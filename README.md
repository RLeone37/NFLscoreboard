🏆 Sports Scoreboard
A fast, mobile-first sports scoreboard web app built for iOS home screen use. Tracks live, upcoming, and completed games across NFL, MLB, and NHL — all from a single lightweight HTML file with no dependencies or build tools required.

📱 Features
	•	Three sports in one — NFL, MLB, and NHL via a bottom navigation bar
	•	Live / Upcoming / Final tabs — filter games by status across all sports
	•	Auto-refreshes every 30 seconds — no manual reload needed
	•	Sport-specific game details:
	•	🏈 NFL — score, quarter, game clock, down & distance, possession indicator, timeout dots
	•	⚾ MLB — score, inning (top/bottom), count, outs, on-base diamond, full linescore (R/H/E), probable starters
	•	🏒 NHL — score, period, game clock, shots on goal, OT/SO indicators, probable goalies
	•	Team logos — uses local logos with automatic ESPN CDN fallback
	•	Season records displayed under each team
	•	iOS PWA ready — add to home screen for a native app-like experience
	•	Live badge indicators — red dot appears on any sport tab with an active game
